### Métodos http 

**@GetMapping**	Obtener datos
**@PostMapping** Crear nuevos recursos
**@PutMapping**	Actualizar (reemplazar)
**@DeleteMapping**	Eliminar recursos
**@PatchMapping**	Actualización parcial
**(Todos) @RequestMapping**	Uso más genérico
